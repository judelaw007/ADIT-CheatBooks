# 02. Major Question Types

## Purpose
This is the core of the CheatBook - template-driven frameworks for every major question type you'll encounter in the ADIT UK exam. Each template includes structure, key considerations, and worked examples from past papers.

## Question Type Organization

### Part A Questions (25 marks each - 45 minutes per question)
**Characteristics**:
- Complex, multi-layered scenarios
- Require comprehensive analysis
- Often combine multiple tax areas
- Demand structured, professional presentation

**Topics**:
- Residence (Corporate and Individual)
- Permanent Establishment
- Remittance Basis

### Part B Questions (20 marks - 36 minutes per question)
**Characteristics**:
- Focus on anti-avoidance provisions
- International tax implications
- Require knowledge of specific legislative provisions
- Often involve calculations

**Topics**:
- Controlled Foreign Companies (CFCs)
- Thin Capitalization
- Transfer Pricing
- Business Investment Relief

### Part C Questions (15 marks each - 27 minutes per question)
**Characteristics**:
- More focused, specific topics
- Shorter, targeted answers required
- May include quick calculations
- Often test application of recent legislation

**Topics**:
- Domicile and deemed domicile
- Trusts taxation
- ATED (Annual Tax on Enveloped Dwellings)
- Withholding Tax
- Other miscellaneous topics

## Template Structure

Each topic folder contains:

1. **TEMPLATE.md**: Master template with question framework
   - Recognition triggers (how to identify this question type)
   - Core structure/outline
   - Key legislation references
   - Common variations
   - Time allocation within the question

2. **Worked_Examples.md**: Past paper questions with solutions
   - Annotated solutions showing template application
   - Marking scheme breakdown
   - Common errors to avoid
   - Examiner's preferred approach

3. **Key_Points.md**: Critical knowledge for this topic
   - Essential rules and definitions
   - Numerical thresholds and limits
   - Recent legislative changes
   - Interaction with other topics

4. **Practice_Drills.md**: Focused exercises
   - Quick-fire questions
   - Calculation practice
   - Scenario variations

## How to Use This Section

### Phase 1: Template Learning (Week 3-4)
1. Read TEMPLATE.md for one topic
2. Study one worked example
3. Attempt a second example independently
4. Compare your answer to the solution

### Phase 2: Template Mastery (Week 5-6)
1. Practice timed answers (strict time limits)
2. Work through all examples in each topic
3. Identify your weak areas
4. Create personal shortcuts and mnemonics

### Phase 3: Integration (Week 7-8)
1. Mix questions from different topics
2. Practice identifying question types quickly
3. Simulate exam conditions with full papers

## Time Allocation Strategy

### Part A (25 marks = 45 minutes)
- Read and analyze: 5 minutes
- Plan structure: 3 minutes
- Write answer: 34 minutes
- Review: 3 minutes

### Part B (20 marks = 36 minutes)
- Read and analyze: 4 minutes
- Plan structure: 2 minutes
- Write answer: 27 minutes
- Review: 3 minutes

### Part C (15 marks = 27 minutes)
- Read and analyze: 3 minutes
- Plan structure: 2 minutes
- Write answer: 20 minutes
- Review: 2 minutes
