| First AUT | ≥183 days in UK | ⬜ |

### Sufficient Ties Test
| Prior Residence | Days in UK | Ties Needed | Verified |
|-----------------|------------|-------------|----------|
| UK resident in any of prior 3 years | ≥16 days | - | ⬜ |
| | 16-45 days | 4+ ties | ⬜ |
| | 46-90 days | 3+ ties | ⬜ |
| | 91-120 days | 2+ ties | ⬜ |
| | 121+ days | 1+ tie | ⬜ |
| Not UK resident in prior 3 years | ≥46 days | - | ⬜ |
| | 46-90 days | 4 ties (all) | ⬜ |
| | 91-120 days | 3+ ties | ⬜ |
| | 121-182 days | 2+ ties | ⬜ |

### The Five UK Ties
1. Family tie (spouse/civil partner/minor child UK resident)
2. Accommodation tie (available ≥91 days, used ≥1 night)
3. Work tie (≥40 days substantive UK employment)
4. 90-day tie (>90 days in UK in either of 2 prior years)
5. Country tie (more midnights in UK than any other country) - leavers only

## 6. Key Formulas

### Income Tax Liability
```
Taxable Income = Total Income - Personal Allowance - Reliefs
IT Liability = Sum of (Each band × applicable rate)
Net IT = IT Liability - Tax Reducers - PAYE deducted
```
Verified: ⬜

### Corporation Tax
```
CT Liability = TTP × CT Rate
Small Profits Relief = (Upper Limit - TTP) × Standard Fraction × (TTP/TTP)
Marginal Rate Formula: Upper limit × (1 - N/TTP) × 3/200
```
Verified: ⬜

### Capital Gains (Individual)
```
Gain = Proceeds - Cost - Enhancement expenditure
Net Gain = Gain - Losses brought forward - Annual Exempt Amount
CGT = Net Gain × applicable rate
```
Verified: ⬜

### Capital Gains (Corporate)
```
Indexed Gain = Proceeds - (Cost × Indexation Factor)
Note: Indexation frozen at December 2017
```
Verified: ⬜

### Inheritance Tax
```
Estate Value = Assets - Liabilities - Exempt transfers
Taxable Estate = Estate Value - NRB - RNRB
IHT = Taxable Estate × 40% (or 36% with 10%+ charity)
```
Verified: ⬜

### CFC Apportionment
```
UK Shareholder's CFC Charge = CFC Chargeable Profits × Relevant Interest %
```
Verified: ⬜

### Transfer Pricing Adjustment
```
UK Tax Adjustment = (Arm's Length Price - Actual Price) × CT Rate
```
Verified: ⬜

### Corporate Interest Restriction
```
Tax-EBITDA = Taxable Profits + Net Tax-Interest + Capital Allowances
Interest Capacity = Tax-EBITDA × 30%
Disallowance = Tax-Interest - max(Fixed Ratio Amount, Group Ratio Amount)
De minimis: £2m net interest expense
```
Verified: ⬜

### Double Tax Relief
```
Credit Relief = Lower of: (UK tax on foreign income, Foreign tax paid)
Max Credit = UK Tax × (Foreign Income / Total Income)
```
Verified: ⬜
