## CONTENT QUALITY STANDARDS

### For Templates
- ✓ Clear step-by-step structure
- ✓ Visual diagrams where helpful
- ✓ Legislation citations for each element
- ✓ Minimum 2-3 worked examples per major topic
- ✓ Common pitfalls highlighted
- ✓ Mark allocation guidance

### For Worked Examples
- ✓ Based on actual past papers
- ✓ Include both question and model answer
- ✓ Show working papers/calculations
- ✓ Annotate with template mapping
- ✓ Timing guidance (actual minutes taken)
- ✓ Mark breakdown

### For Quick References
- ✓ One-page maximum per topic
- ✓ Visual hierarchy (tables, bullets)
- ✓ Easy to scan
- ✓ No narrative explanations
- ✓ Cross-references to main sections
