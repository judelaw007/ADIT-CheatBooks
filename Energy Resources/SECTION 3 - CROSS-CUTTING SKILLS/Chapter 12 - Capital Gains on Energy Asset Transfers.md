## 12.1 DIRECT VS INDIRECT TRANSFER DISTINCTION

### What is a Capital Gains Tax (CGT) Event in Energy Context?

**Definition** (from June 2025 Q4): Capital Gains Tax (CGT) applies when a company **sells oil and gas assets or transfers ownership** in projects. The taxation of such gains depends on domestic tax laws, international tax treaties, and whether the transfer is **direct or indirect**.

### Direct Transfers

**Definition**: A direct transfer occurs when a company sells **tangible oil and gas assets** directly to another party.

**Assets Commonly Transferred Directly**:
- Drilling rigs and equipment
- Pipelines and processing facilities
- Production rights and licenses
- Working interests in fields
- Petroleum reserves in place

```
DIRECT TRANSFER STRUCTURE
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│    SELLER                              BUYER                     │
│    ┌──────────┐                        ┌──────────┐             │
│    │ Company A│  ───────────────────►  │ Company B│             │
│    └──────────┘   Oil & Gas Assets     └──────────┘             │
│         │         (License, Rigs,           │                    │
│         │          Pipelines)               │                    │
│         ▼                                   ▼                    │
│    Receives Cash                       Acquires Assets           │
│    CGT Event                           New Cost Base             │
│                                                                  │
│    TAX IMPLICATIONS:                                             │
│    • Gain = Proceeds - Cost Base                                │
│    • Taxable in jurisdiction where assets located               │
│    • Clear nexus to source country                              │
│    • May trigger transfer taxes/stamp duty                      │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

**Taxation of Direct Transfers**:

| Aspect | Treatment |
|--------|-----------|
| **Taxing jurisdiction** | Country where assets are located |
| **Basis for taxation** | Clear territorial nexus to source country |
| **Treaty protection** | Limited - Article 13(1) gives source state rights |
| **Valuation** | Usually based on transaction value |
| **Administrative** | Straightforward identification |

### Indirect Transfers

**Definition**: An indirect transfer happens when a company sells **shares in an entity that holds oil and gas assets**, rather than selling the assets themselves.

```
INDIRECT TRANSFER STRUCTURE
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│    SELLER                              BUYER                     │
│    (Offshore)                          (Offshore)                │
│    ┌──────────┐                        ┌──────────┐             │
│    │ Company A│  ═══════════════════►  │ Company B│             │
│    └──────────┘   SHARES in HoldCo     └──────────┘             │
│         │                                   │                    │
│         │ owns 100%                         │ now owns 100%      │
│         ▼                                   ▼                    │
│    ┌──────────────────────────────────────────────┐             │
│    │            HOLDING COMPANY                    │             │
│    │         (Intermediate jurisdiction)           │             │
│    └──────────────────────────────────────────────┘             │
│                        │ owns 100%                               │
│                        ▼                                         │
│    ┌──────────────────────────────────────────────┐             │
│    │           OPERATING COMPANY                   │             │
│    │         (Source country - Country X)          │             │
│    │         Holds: License, Rigs, Pipelines      │             │
│    └──────────────────────────────────────────────┘             │
│                                                                  │
│    KEY QUESTION: Can Country X tax this share sale?             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

**Why Indirect Transfers Are Used**:

| Reason | Explanation |
|--------|-------------|
| **Avoid source country CGT** | Many countries don't tax offshore share sales |
| **Avoid transfer taxes** | No direct asset transfer in source country |
| **Preserve licenses** | License stays with same entity - no re-approval needed |
| **Simplify transaction** | Single share purchase vs multiple asset transfers |
| **Maintain contracts** | Operating agreements remain with same company |

**Source Country Response - Tracing Rules**:

Many countries have enacted **domestic tracing rules** (also called "look-through" or "deeming provisions") to tax indirect transfers:

```
COUNTRIES WITH INDIRECT TRANSFER RULES
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  ESTABLISHED RULES:              EMERGING RULES:                 │
│  ─────────────────               ───────────────                 │
│  • India                         • Peru                          │
│  • China                         • Angola                        │
│  • Indonesia                     • Mozambique                    │
│  • Australia                     • Tanzania                      │
│  • Canada                        • Uganda                        │
│  • United States                 • Kenya                         │
│  • Brazil                        • Ghana                         │
│  • Nigeria                       • Senegal                       │
│                                                                  │
│  THRESHOLD TESTS (typical):                                      │
│  • >50% of share value from immovable property/resources        │
│  • >50% of assets in source country                             │
│  • Substantial interest threshold (10-50% ownership)            │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Direct vs Indirect Comparison Matrix

| Factor | Direct Transfer | Indirect Transfer |
|--------|-----------------|-------------------|
| **What is sold** | Physical assets/licenses | Shares in holding company |
| **Source country taxation** | Clear right to tax | May be challenged |
| **Treaty protection** | Article 13(1) - source state | Article 13(4) may apply |
| **License continuity** | Usually requires approval | License remains unchanged |
| **Transfer taxes** | Often applicable | Usually avoided |
| **Complexity** | Lower | Higher |
| **Anti-avoidance risk** | Low | High if structure aggressive |

## 12.2 IMMOVABLE PROPERTY PROVISIONS (ARTICLE 13(4))

### OECD Model Article 13 - Capital Gains

**Article 13 Hierarchy**:

```
OECD MODEL ARTICLE 13 - CAPITAL GAINS
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  Art 13(1): IMMOVABLE PROPERTY                                   │
│  ─────────────────────────────                                   │
│  Gains from alienation of immovable property situated in        │
│  other State → MAY BE TAXED in that other State                 │
│  (Source state has taxing rights)                               │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Art 13(2): BUSINESS PROPERTY OF PE                              │
│  ───────────────────────────────────                             │
│  Gains from movable property of PE → Source state taxation       │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Art 13(3): SHIPS AND AIRCRAFT                                   │
│  ──────────────────────────────                                  │
│  → Residence state only                                          │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Art 13(4): SHARES IN IMMOVABLE PROPERTY COMPANIES               │
│  ─────────────────────────────────────────────────               │
│  Gains from shares deriving >50% value from immovable           │
│  property → MAY BE TAXED in source state                        │
│  **CRITICAL FOR ENERGY TRANSACTIONS**                           │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  Art 13(5): OTHER PROPERTY                                       │
│  ──────────────────────────                                      │
│  All other gains → Residence state only (default rule)          │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Article 13(4) - The Immovable Property Shares Provision

**OECD Model Text**:
> "Gains derived by a resident of a Contracting State from the alienation of shares... deriving more than 50 per cent of their value directly or indirectly from immovable property situated in the other Contracting State may be taxed in that other State."

**Key Elements**:

| Element | Interpretation |
|---------|----------------|
| **"more than 50 per cent"** | Threshold test - must exceed 50% |
| **"value"** | Fair market value, not book value |
| **"directly or indirectly"** | Looks through holding structures |
| **"immovable property"** | Includes mineral rights - see below |
| **"may be taxed"** | Non-exclusive right - source state CAN tax |

### What Constitutes "Immovable Property" for Energy

**Critical Definition** (OECD Commentary + domestic law):

```
IMMOVABLE PROPERTY IN ENERGY CONTEXT
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  CLEARLY INCLUDED:                                               │
│  ─────────────────                                               │
│  • Land and buildings                                            │
│  • Mineral deposits and reserves                                │
│  • Oil and gas reservoirs                                       │
│  • Production licenses and exploration permits                   │
│  • Mining rights and concessions                                │
│  • Rights to exploit natural resources                          │
│  • Petroleum agreements (PSCs, concessions)                     │
│                                                                  │
│  UK SPECIFIC (June 2025 Q4):                                     │
│  "Immovable property includes the licence interest and the      │
│   oil reservoir"                                                │
│                                                                  │
│  AUSTRALIA SPECIFIC:                                             │
│  Principal Asset Test includes:                                  │
│  • Mining, quarrying or prospecting rights                      │
│  • Leases over mining areas                                     │
│  • Interests in petroleum titles                                │
│                                                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  MAY BE EXCLUDED:                                                │
│  ────────────────                                                │
│  • Drilling equipment (movable)                                 │
│  • Vessels and FPSOs (potentially movable)                      │
│  • Pipelines (depends on jurisdiction)                          │
│  • Technical know-how and IP                                    │
│  • Working capital and receivables                              │
│  • Cash and financial instruments                               │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### The 50% Value Test

**Calculation Framework**:

```
50% IMMOVABLE PROPERTY VALUE TEST
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  STEP 1: Identify All Assets                                     │
│  ─────────────────────────────                                   │
│  List all assets of target company at fair market value         │
│                                                                  │
│  STEP 2: Classify Assets                                         │
│  ───────────────────────                                         │
│  Immovable property:                                            │
│  • Petroleum license:          $200M                            │
│  • Reserves in place:          $150M                            │
│  • Production facilities:       $80M                            │
│  • Land and buildings:          $20M                            │
│  SUBTOTAL IMMOVABLE:           $450M                            │
│                                                                  │
│  Other assets:                                                  │
│  • Cash and receivables:        $50M                            │
│  • Equipment (movable):         $30M                            │
│  • Working capital:             $20M                            │
│  SUBTOTAL OTHER:               $100M                            │
│                                                                  │
│  TOTAL ASSETS:                 $550M                            │
│                                                                  │
│  STEP 3: Calculate Percentage                                    │
│  ────────────────────────────                                    │
│  Immovable / Total = $450M / $550M = 81.8%                      │
│                                                                  │
│  RESULT: >50% threshold met                                      │
│  → Source country MAY tax the share sale under Article 13(4)   │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Timing of the 50% Test

**Issue**: When is the 50% test applied?

```
TIMING CONSIDERATIONS
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  OECD MODEL APPROACH:                                            │
│  • At time of disposal                                          │
│  • Some treaties: at any time during preceding 365 days         │
│                                                                  │
│  AUSTRALIA 2024-25 BUDGET UPDATE:                                │
│  • 365-day testing period for Principal Asset Test              │
│  • Prevents "asset stuffing" immediately before disposal        │
│  • Looks back over full year                                    │
│                                                                  │
│  ANTI-AVOIDANCE CONCERN:                                         │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │ Company injects cash before sale:                            ││
│  │ Before:  Immovable $60M / Total $100M = 60% (taxable)       ││
│  │ After:   Immovable $60M / Total $130M = 46% (not taxable)   ││
│  │ → 365-day lookback prevents this manipulation               ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Treaty Variations

**Not all treaties follow OECD Model exactly**:

| Treaty Variation | Effect |
|-----------------|--------|
| **No Article 13(4)** | Gains on shares taxable only in residence state |
| **Higher threshold (75%)** | Harder to meet - less source state taxation |
| **Lower threshold (25%)** | Easier to meet - more source state taxation |
| **365-day lookback** | Anti-avoidance protection |
| **"Principal value" test** | Similar but different wording |
| **Participation threshold** | May require minimum % ownership to apply |

## 12.3 DOMESTIC SOURCING RULES

### Source Country Taxing Rights

**General Principle**: Even where a tax treaty exists, domestic law must first give a country the right to tax before treaty provisions become relevant.

```
DOMESTIC LAW → TREATY INTERACTION
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  STEP 1: Does domestic law create a taxing right?               │
│          ↓                                                       │
│          YES → Proceed to Step 2                                │
│          NO  → No tax (regardless of treaty)                    │
│                                                                  │
│  STEP 2: Is there an applicable tax treaty?                     │
│          ↓                                                       │
│          YES → Treaty may LIMIT (not expand) domestic right     │
│          NO  → Full domestic taxation applies                   │
│                                                                  │
│  STEP 3: What does the treaty provide?                          │
│          ↓                                                       │
│          Article 13(4) applies → Source CAN tax                 │
│          Article 13(5) applies → Only residence state taxes     │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 12.4 COST BASE AND INDEXATION CONSIDERATIONS

### Determining Cost Base

**Definition**: The cost base is the amount against which sale proceeds are compared to calculate the capital gain.

```
CAPITAL GAIN CALCULATION
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  BASIC FORMULA:                                                  │
│                                                                  │
│  Capital Gain = Sale Proceeds - Cost Base                       │
│                                                                  │
│  WHERE COST BASE INCLUDES:                                       │
│  ───────────────────────────                                     │
│  1. Original acquisition cost                                   │
│  2. Plus: Incidental acquisition costs                          │
│     • Legal fees                                                │
│     • Due diligence costs                                       │
│     • Stamp duty                                                │
│  3. Plus: Enhancement expenditure                               │
│     • Exploration costs                                         │
│     • Development capex                                         │
│     • Appraisal expenditure                                     │
│  4. Plus: Costs of establishing title                           │
│  5. Less: Depreciation/amortization claimed                     │
│  6. Plus: Indexation allowance (if applicable)                  │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Cost Base Components in Energy Transactions

| Component | Treatment | Considerations |
|-----------|-----------|----------------|
| **Acquisition cost** | Original purchase price | May be allocated across multiple assets |
| **Exploration costs** | Usually capitalized | May be expensed for income tax |
| **Development capex** | Added to base | Subject to depreciation rules |
| **Seismic data** | May be capitalized | Depends on jurisdiction |
| **Signature bonuses** | Part of license cost | Usually fully capitalized |
| **Carried costs** | Complex allocation | Depend on farm-in terms |
| **Currency translation** | Historical vs current | Can significantly affect gain |

### Indexation Allowances

**Purpose**: To ensure only real gains (above inflation) are taxed.

```
INDEXATION EXAMPLE
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  SCENARIO: License acquired in 2015, sold in 2025               │
│                                                                  │
│  Original cost (2015):                    $100,000,000          │
│  Cumulative inflation (2015-2025):        35%                   │
│  Indexed cost base:                       $135,000,000          │
│                                                                  │
│  Sale proceeds (2025):                    $200,000,000          │
│                                                                  │
│  WITHOUT INDEXATION:                                             │
│  Gain = $200M - $100M = $100M                                   │
│  Tax @ 30% = $30M                                               │
│                                                                  │
│  WITH INDEXATION:                                                │
│  Gain = $200M - $135M = $65M                                    │
│  Tax @ 30% = $19.5M                                             │
│                                                                  │
│  INDEXATION BENEFIT: $10.5M tax saving                          │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Jurisdictional Variations

| Jurisdiction | Indexation Treatment |
|-------------|---------------------|
| **UK** | No indexation for corporate gains (from 2018) |
| **Australia** | Indexation frozen at 1999; 50% discount alternative |
| **Canada** | 50% inclusion rate (effectively 50% discount) |
| **US** | No inflation indexation; holding period affects rate |
| **Norway** | Generally taxed at ordinary rates; no indexation |
| **Many resource countries** | Full gain taxed; no indexation relief |

### Farm-In Cost Base Issues

**Special Consideration**: When interests are acquired through farm-in arrangements, cost base determination can be complex.

```
FARM-IN COST BASE DETERMINATION
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  SCENARIO: Company B farms in to 50% of Company A's license     │
│                                                                  │
│  Terms:                                                          │
│  • B pays 100% of next $40M exploration costs                   │
│  • B receives 50% working interest                              │
│  • No cash changes hands                                        │
│                                                                  │
│  COST BASE QUESTIONS:                                            │
│                                                                  │
│  For Company A (farmer-out):                                     │
│  • Did A dispose of 50% for market value?                       │
│  • What is A's remaining cost base in 50%?                      │
│  • Is there a deemed disposal?                                  │
│                                                                  │
│  For Company B (farmer-in):                                      │
│  • Is B's cost base $40M (actual spend)?                        │
│  • Or is it market value of 50% interest?                       │
│  • How are carried costs treated?                               │
│                                                                  │
│  PLANNING POINT:                                                 │
│  Structure matters - characterization as purchase, joint        │
│  venture contribution, or service arrangement affects tax       │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Valuation Methods

**From June 2025 Q4**: Multiple valuation methods may be relevant for CGT purposes:

| Method | Application | Energy Context |
|--------|-------------|----------------|
| **DCF (Discounted Cash Flow)** | Primary for producing assets | NPV of future production cash flows |
| **Comparable Transactions** | Market-based | Recent similar asset sales |
| **Cost-Based** | Exploration assets | Accumulated exploration expenditure |
| **NAV (Net Asset Value)** | Holding companies | Sum of underlying asset values |
| **Replacement Cost** | Infrastructure | Cost to replicate facilities |

## 12.5 ANTI-AVOIDANCE PROVISIONS

### Treaty Shopping and CGT

**Issue**: Taxpayers may interpose holding companies in favorable treaty jurisdictions to avoid source country CGT.

```
TREATY SHOPPING EXAMPLE
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  STRUCTURE WITHOUT TREATY PLANNING:                              │
│                                                                  │
│  US Parent ──────sells shares──────► Buyer                       │
│       │                                                          │
│       │ owns 100%                                                │
│       ▼                                                          │
│  Country X OpCo (License)                                        │
│                                                                  │
│  Result: Country X taxes gain under domestic law                 │
│          No treaty protection (US-X treaty has Art 13(4))       │
│                                                                  │
│  ─────────────────────────────────────────────────────────────  │
│                                                                  │
│  STRUCTURE WITH TREATY PLANNING:                                 │
│                                                                  │
│  US Parent                                                       │
│       │ owns 100%                                                │
│       ▼                                                          │
│  Netherlands HoldCo ───sells shares───► Buyer                    │
│       │                                                          │
│       │ owns 100%                                                │
│       ▼                                                          │
│  Country X OpCo (License)                                        │
│                                                                  │
│  Intended result: NL-X treaty has NO Art 13(4)                   │
│                   Country X cannot tax the gain                  │
│                                                                  │
│  BUT: Anti-avoidance provisions may deny treaty benefits        │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### BEPS Action 6 - Preventing Treaty Abuse

**From June 2025 Q4**: "BEPS Action 6 addresses treaty abuse, and countries are implementing measures like the Principal Purpose Test (PPT) and Limitation on Benefits (LOB) clauses."

```
ANTI-ABUSE PROVISIONS
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │

### Example 1: Direct vs Indirect Transfer Analysis (June 2025 Q4 Style - 25 marks)

**Question**: Explain the distinction between direct and indirect transfers in the oil and gas context, including the tax implications and how tax treaties affect the taxation of capital gains.

**Model Answer**:

```
PART 1: DIRECT VS INDIRECT TRANSFERS (8 marks)

DIRECT TRANSFERS
────────────────
Definition:
A direct transfer occurs when a company sells tangible oil and gas
assets such as drilling rigs, pipelines, or production rights
directly to another party.

Key characteristics:
• Assets physically transferred
• Clear nexus to source country
• License assignment typically required
• Government approval often needed
• Transfer taxes may apply

Tax treatment:
• Gain calculated as proceeds minus cost base
• Taxable in jurisdiction where assets located
• Treaty Article 13(1) gives source state taxing rights
• Limited scope for tax planning

INDIRECT TRANSFERS
──────────────────
Definition:
An indirect transfer happens when a company sells shares in an
entity that holds oil and gas assets, rather than selling the
assets themselves.

Key characteristics:
• Shares sold, not underlying assets
• Transaction occurs offshore
• License remains with same entity
• No government approval typically required
• Avoids transfer taxes

Tax treatment:
• Many countries now impose CGT on indirect transfers
• Threshold test: typically >50% value from immovable property
• Treaty Article 13(4) may preserve source state rights
• Structuring opportunities through holding companies

PART 2: TAX TREATY ANALYSIS (10 marks)

OECD MODEL ARTICLE 13 FRAMEWORK
───────────────────────────────
Treaties based on the OECD Model Convention generally assign
taxing rights as follows:

Article 13(1) - Immovable Property:
• Gains from direct sales of immovable property
• Source state HAS taxing rights
• Applies to direct asset transfers

Article 13(4) - Shares in Immovable Property Companies:
• Gains from shares deriving >50% value from immovable property
• Source state MAY tax if threshold met
• Key provision for indirect transfers
• Immovable property INCLUDES mineral rights, petroleum licenses

Article 13(5) - Residual Provision:
• All other gains taxable ONLY in residence state
• Applies if <50% value from immovable property
• Creates planning opportunities

APPLICATION TO OIL & GAS:
─────────────────────────
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  Question: Does share sale trigger source country tax?          │
│                                                                  │
│  Step 1: Calculate immovable property percentage                │
│          Petroleum license value:      $200M                    │
│          Reserves value:               $150M                    │
│          Other (cash, equipment):       $50M                    │
│          Total:                        $400M                    │
│          Immovable %: 87.5%                                     │
│                                                                  │
│  Step 2: Apply Article 13(4)                                    │
│          >50% threshold MET                                      │
│          Source state CAN tax the share sale                    │
│                                                                  │
│  Step 3: Check domestic law                                     │
│          Does source country have enabling legislation?         │
│          If yes: tax applies                                    │
│          If no: no tax despite treaty permission                │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

PART 3: ANTI-AVOIDANCE CONSIDERATIONS (7 marks)

BEPS ACTION 6 MEASURES
──────────────────────
BEPS Action 6 addresses treaty abuse. Countries implement:

1. Principal Purpose Test (PPT):
   Treaty benefit denied if obtaining the benefit was one of
   the principal purposes of the arrangement

2. Limitation on Benefits (LOB):
   Objective tests for treaty entitlement:
   • Publicly traded company test
   • Ownership and base erosion test
   • Active trade or business test

APPLICATION:

│                                                                  │

## 12.8 ERROR PREVENTION

### Common Exam Mistakes

| Error | Why It's Wrong | Correct Approach |
|-------|---------------|------------------|
| Assuming all share sales avoid source CGT | Many countries tax indirect transfers | Check domestic tracing rules |
| Ignoring Article 13(4) threshold test | Must calculate immovable property % | Always compute the 50% test |
| Forgetting domestic law comes first | Treaty can only limit, not create tax | Analyze domestic law then treaty |
| Overlooking anti-avoidance provisions | PPT/LOB may deny treaty benefits | Assess anti-abuse risk |
| Confusing immovable property scope | Includes mineral rights and licenses | Define what's included in source country |
| Missing cost base adjustments | Cost base includes various elements | Calculate properly including enhancements |
| Ignoring timing of 50% test | Lookback periods may apply | Check if 365-day rule applies |
| Not mentioning Shell Australia | Key illustrative case | Reference for indirect transfer risks |

## 12.10 CHAPTER SUMMARY

### Key Takeaways

1. **Direct transfers** involve sale of physical assets; **indirect transfers** involve sale of shares in asset-holding entities
2. **Article 13(4)** allows source country taxation where >50% of share value derives from immovable property
3. **Immovable property** for energy includes petroleum licenses, mineral rights, and oil reservoirs
4. Many countries have **domestic tracing rules** to tax indirect transfers (Peru, Angola, Mozambique)
5. **Anti-avoidance provisions** (PPT, LOB, GAAR) may deny treaty benefits for aggressive structures
6. **Cost base** includes acquisition cost, exploration expenditure, and enhancement costs
7. **Shell Australia case** ($755M settlement) demonstrates significant CGT enforcement risk
8. **Due diligence** must address historical transactions, holding structures, and exit planning
