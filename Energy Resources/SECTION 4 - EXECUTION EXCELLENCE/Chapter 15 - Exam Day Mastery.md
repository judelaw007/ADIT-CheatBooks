## 15.3 ANSWER PRESENTATION EXCELLENCE

### Structure and Formatting Standards

**From Exam Instructions**: "Marks may be allocated for clarity of presentation of your answers."

```
ANSWER STRUCTURE TEMPLATE
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│ QUESTION [X] - Part [Y]                                          │
│                                                                  │
│ [Opening sentence directly addressing the question requirement] │
│                                                                  │
│ 1. FIRST MAIN POINT                                              │
│    • Supporting detail                                           │
│    • Example or application                                      │
│                                                                  │
│ 2. SECOND MAIN POINT                                             │
│    • Supporting detail                                           │
│    • Example or application                                      │
│                                                                  │
│ [Continue for each distinct point]                               │
│                                                                  │
│ [Concluding statement or recommendation if appropriate]         │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Formatting Best Practices

| Element | Best Practice | Example |
|---------|---------------|---------|
| **Headings** | Use for each question part | "Part 1: Key Features of PSAs" |
| **Numbering** | Number distinct points | "1. Cost Oil Mechanism..." |
| **Paragraphs** | One idea per paragraph | Short, focused paragraphs |
| **Line breaks** | Between questions and parts | Clear visual separation |
| **Bold text** | For key terms (if available) | "**Transfer pricing**" |
| **Calculations** | Show all workings clearly | Step-by-step presentation |

### Mark-Maximizing Techniques

```
MARKS PER MINUTE STRATEGY
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│ GOLDEN RULE: 1.8-2.0 minutes per mark                           │
│                                                                  │
│ Part A (25 marks each):  45-48 minutes per question             │
│ Part B (20 marks):       36-39 minutes                          │
│ Part C (15 marks each):  27-29 minutes per question             │
│ Reading/Planning:        10-15 minutes                          │
│                                                                  │
│ TOTAL:                   195 minutes                            │
│                                                                  │
│ POINT-SCORING APPROACH:                                          │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ • Aim for 1 distinct point = 1-2 marks                      │ │
│ │ • For 15-mark question: need 8-10 good points               │ │
│ │ • For 20-mark question: need 10-12 good points              │ │
│ │ • For 25-mark question: need 12-15 good points              │ │
│ │                                                              │ │
│ │ BREADTH over DEPTH (usually):                               │ │
│ │ • Better to make 10 points briefly than 5 in detail         │ │
│ │ • Move on after making your point                           │ │
│ │ • Diminishing returns on extensive elaboration              │ │
│ └─────────────────────────────────────────────────────────────┘ │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Calculation Presentation

**From Exam Instructions**: "All workings should be made in appropriate monetary currency... to the nearest whole unit."

```
CALCULATION PRESENTATION FORMAT
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│ EXAMPLE: Government Take Calculation                             │
│                                                                  │
│ Step 1: Calculate Gross Revenue                                  │
│ Production: 1,000,000 barrels × $50/barrel = $50,000,000        │
│                                                                  │
│ Step 2: Deduct Royalty                                           │
│ Royalty: $50,000,000 × 10% = $5,000,000                         │
│ Revenue after royalty: $50,000,000 - $5,000,000 = $45,000,000   │
│                                                                  │
│ Step 3: Calculate Cost Oil                                       │
│ Recoverable costs: $15,000,000                                   │
│ Cost oil = $15,000,000                                           │
│                                                                  │
│ Step 4: Calculate Profit Oil                                     │
│ Profit oil: $45,000,000 - $15,000,000 = $30,000,000             │
│                                                                  │
│ Step 5: Split Profit Oil                                         │
│ Government share (60%): $30,000,000 × 60% = $18,000,000         │
│ Contractor share (40%): $30,000,000 × 40% = $12,000,000         │
│                                                                  │
│ TOTAL GOVERNMENT TAKE:                                           │
│ Royalty:           $5,000,000                                    │
│ Profit oil share:  $18,000,000                                   │
│ Corporate tax:     [calculated separately]                       │
│ ─────────────────────────────                                   │
│ TOTAL:             $[sum]                                        │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

KEY PRINCIPLES:
• Label each step clearly
• Show the formula, then the numbers, then the answer
• Use consistent currency throughout (£ or $ as given)
• Round to whole units as instructed
• Double-check arithmetic before moving on
```

## 15.4 TIME RECOVERY TACTICS

### When Falling Behind

```
TIME CRISIS PROTOCOLS
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│ SCENARIO 1: 10+ minutes behind at Part A completion             │
│ ───────────────────────────────────────────────────────────────│
│ ACTION:                                                          │
│ • Reduce Part B allocation to 30-32 minutes                     │
│ • Focus on high-value points only                               │
│ • Use bullet points instead of full sentences                   │
│ • Skip detailed examples - state the principle                  │
│                                                                  │
│ SCENARIO 2: Running out of time on current question             │
│ ───────────────────────────────────────────────────────────────│
│ ACTION:                                                          │
│ • List remaining points as bullets (1 mark each)                │
│ • Move to next question                                          │
│ • Partial answers score marks; blank pages score zero           │
│                                                                  │
│ SCENARIO 3: 15+ minutes remaining, all questions answered       │
│ ───────────────────────────────────────────────────────────────│
│ ACTION:                                                          │
│ • Return to weakest answer                                       │
│ • Add one additional point to each Part C answer                │
│ • Check calculations for errors                                  │
│ • Ensure all question parts are addressed                       │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Time Trap Warning Signs

| Warning Sign | What's Happening | Recovery Action |
|--------------|------------------|-----------------|
| Writing for 5+ minutes without a new point | Over-elaborating | Move to next point |
| Re-reading question multiple times | Confusion/panic | Write what you know; refine later |
| Stuck on one calculation step | Arithmetic block | Move on; mark to return |
| Perfectionism on wording | Time leak | Accept "good enough" |
| Part A taking 60+ minutes total | Severe overrun | Shift to bullet points |

### Emergency Bullet Point Protocol

**Use when severely time-constrained (final 20 minutes with questions incomplete)**

```
EMERGENCY ANSWER FORMAT
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│ Question [X] - [Topic]                                           │
│                                                                  │
│ Key points:                                                      │
│ • [Point 1 - stated as briefly as possible]                     │
│ • [Point 2]                                                      │
│ • [Point 3]                                                      │
│ • [Point 4]                                                      │
│ • [Point 5]                                                      │
│                                                                  │
│ [Move immediately to next question]                              │
│                                                                  │
│ RATIONALE:                                                       │
│ • 5 bullet points = potentially 5+ marks                        │
│ • 5 marks from bullets > 0 marks from blank page               │
│ • Examiner awards marks for knowledge demonstrated              │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Decoding Question Requirements

```
QUESTION REQUIREMENT KEYWORDS
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│ KEYWORD          │ WHAT IT REQUIRES                              │
│ ─────────────────┼──────────────────────────────────────────────│
│ "Discuss"        │ Explore multiple aspects; pros and cons      │
│ "Explain"        │ Describe how/why something works             │
│ "Advise"         │ Give recommendations with reasoning          │
│ "Calculate"      │ Show workings and numerical answer           │
│ "Compare"        │ Identify similarities AND differences        │
│ "Critically      │ Evaluate strengths and weaknesses;           │
│  evaluate"       │ make a judgment                              │
│ "Outline"        │ Brief overview; main points only             │
│ "Identify"       │ List/name items (brief)                      │
│ "Analyse"        │ Break down into components; examine closely  │
│ "Define"         │ Give precise meaning/description             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Multi-Part Question Strategy

**From Past Papers**: Most questions have 2-3 parts with separate mark allocations.

```
MULTI-PART QUESTION APPROACH
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│ EXAMPLE (June 2025 Q1):                                          │
│ 1) Advise on key features of concession agreements and PSAs (16)│
│ 2) Explain which form offers highest revenue (4)                │
│ 3) Outline other fiscal revenues or obligations (5)             │
│                                                                  │
│ TIME ALLOCATION:                                                 │
│ Part 1: 16 marks × 1.8 min = ~29 minutes                        │
│ Part 2: 4 marks × 1.8 min = ~7 minutes                          │
│ Part 3: 5 marks × 1.8 min = ~9 minutes                          │
│ Total: 45 minutes                                               │
│                                                                  │
│ CRITICAL RULE:                                                   │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ Answer ALL parts of EVERY question you attempt              │ │
│ │                                                              │ │
│ │ A complete answer with brief parts scores better than       │ │
│ │ an incomplete answer with one detailed part                 │ │
│ │                                                              │ │
│ │ • Part 1 only (excellent): ~12-14 marks                     │ │
│ │ • All 3 parts (adequate): ~15-18 marks                      │ │
│ └─────────────────────────────────────────────────────────────┘ │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 15.6 PROFESSIONAL WRITING STANDARDS FOR TAX ADVICE

### Advisory Memo Format

**When question asks you to "advise"**:

```
ADVISORY ANSWER STRUCTURE
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│ OPENING:                                                         │
│ "In advising [Country/Company] on [topic], the following        │
│ considerations are relevant..."                                  │
│                                                                  │
│ BODY:                                                            │
│ [Structured analysis using numbered points]                      │
│                                                                  │
│ RECOMMENDATION (if applicable):                                  │
│ "Based on the above analysis, [recommendation] because          │
│ [key reasons]."                                                  │
│                                                                  │
│ EXAMPLE OPENING:                                                 │
│ "In advising Country ABC on the choice between a concession     │
│ agreement and a production sharing agreement (PSA), the         │
│ following key features and considerations should be examined..."│
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Professional Language Standards

| Avoid | Use Instead |
|-------|-------------|
| "I think..." | "It is advisable that..." |
| "You should..." | "The company should consider..." |
| "Obviously..." | "It follows that..." |
| Slang or contractions | Formal language |
| "Basically..." | [Remove - add no value] |
| Emotional language | Objective analysis |

### Balanced Analysis Format

**When question asks you to "critically evaluate" or "discuss"**:

```
BALANCED ANALYSIS STRUCTURE
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│ INTRODUCTION:                                                    │
│ "[Topic] involves [brief definition/context]."                  │
│                                                                  │
│ ADVANTAGES / ARGUMENTS FOR:                                      │
│ 1. [First advantage]                                             │
│ 2. [Second advantage]                                            │
│ 3. [Third advantage]                                             │
│                                                                  │
│ DISADVANTAGES / ARGUMENTS AGAINST:                               │
│ 1. [First disadvantage]                                          │
│ 2. [Second disadvantage]                                         │
│ 3. [Third disadvantage]                                          │
│                                                                  │
│ CONCLUSION:                                                      │
│ "On balance, [conclusion] taking into account [key factors]."   │
│                                                                  │
│ EXAMPLE FROM PAST PAPERS (Carbon Tax vs ETS):                    │
│ "While carbon taxes offer price certainty and administrative    │
│ simplicity, ETS provides environmental certainty through        │
│ emission caps. The optimal choice depends on policy priorities  │
│ and the specific characteristics of the jurisdiction."          │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Example 3: Answer Presentation Excellence

**Question**: "Discuss the tax implications of interest deductions in the case of purchasing assets or target shares in oil and gas companies." (10 marks)

```
MODEL ANSWER STRUCTURE

INTEREST DEDUCTIONS IN OIL & GAS M&A

Introduction:
Interest deductions in acquisition financing present different
implications depending on whether assets or shares are purchased.

1. ASSET PURCHASE - INTEREST DEDUCTIONS
   • Interest generally deductible against profits generated by
     acquired assets
   • Assets will produce taxable income, supporting interest
     deductibility
   • Deduction available in the jurisdiction where assets are
     located
   • No participation exemption issues arise

2. SHARE PURCHASE - INTEREST DEDUCTIONS
   • Deductibility may be restricted where dividends from target
     are tax-exempt under participation exemption
   • Interest incurred to acquire shares producing exempt income
     may be disallowed
   • Need to consider domestic rules on interest attribution

3. TAX CONSOLIDATION OPPORTUNITIES
   • If jurisdiction permits tax grouping, interest deductions
     from acquisition vehicle can be offset against target profits
   • Enables value extraction from interest expense
   • Requires meeting consolidation requirements

4. PSC REGIME CONSIDERATIONS
   • PSC agreements typically exclude financing costs from cost oil
   • Interest on debt to acquire PSC interests may need to be
     claimed elsewhere in the multinational group
   • Different treatment from tax/royalty regimes

5. THIN CAPITALISATION RESTRICTIONS
   • Related-party debt subject to thin cap limitations
   • May restrict or limit interest deductions
   • Need to structure debt within safe harbour ratios

CONCLUSION:
The deductibility of acquisition interest depends on the
transaction structure, target jurisdiction rules, and the
availability of tax consolidation. Asset purchases generally
offer clearer deduction paths, while share purchases require
careful analysis of participation exemption interactions.

[Answer demonstrates: clear structure, numbered points,
professional language, technical accuracy, conclusion]
```

## 15.9 ERROR PREVENTION

### Common Exam Day Mistakes

| Mistake | Impact | Prevention |
|---------|--------|------------|
| **Not reading all questions first** | Poor question selection | Invest 10-15 minutes reading |
| **Spending too long on Q1** | Insufficient time for later questions | Set hard time limits |
| **Not answering all parts** | Lost easy marks | Check all parts before moving on |
| **Illegible/disorganized answers** | Examiner frustration, lost marks | Use clear structure, headings |
| **Calculation errors** | Wrong answers, cascading errors | Check arithmetic; state assumptions |
| **Writing everything you know** | Time wasted, unfocused | Answer the question asked |
| **Panicking when stuck** | Poor decisions, time loss | Use bullet points, move on |
| **Not using technical terms** | Appears unprofessional | Use correct terminology |
| **Ignoring mark allocation** | Misallocated effort | Time proportional to marks |
| **No conclusion/recommendation** | Incomplete answer | Always close advisory answers |
