### Why Interest Deductions Matter in Oil & Gas

| Factor | Impact | Exam Relevance |
|--------|--------|----------------|
| **Capital Intensive** | Projects require $billions in financing | High leverage = significant interest expense |
| **Long Payback Periods** | 10-30 year project cycles | Interest accumulates over many years |
| **Ring-Fencing** | Many countries isolate project economics | Interest deductions may be restricted per-project |
| **PSC Complexities** | Cost recovery mechanisms | Financing costs often excluded from cost oil |
| **Tax Arbitrage** | Cross-border structures | Opportunity for interest stripping |

### Interest Deductibility: Tax/Concession vs PSC Regimes

| Aspect | Tax/Concession Regime | Production Sharing Contract |
|--------|----------------------|----------------------------|
| **General Rule** | Interest deductible as business expense | Financing costs typically excluded from cost oil |
| **Acquisition Debt** | Usually deductible | Rarely recoverable as cost oil |
| **Third-Party Debt** | Subject to thin cap rules | May be allowed if specifically permitted in PSC |
| **Related-Party Debt** | Subject to thin cap + arm's length | Generally prohibited or heavily restricted |
| **Timing** | Deductible when incurred | Cost recovery spread over production |

### OECD BEPS Action 4: The Global Standard

> **Key Recommendation**: Limit net interest deductions to a fixed ratio of EBITDA (10-30% corridor, with 30% as the benchmark).

| Element | Description | Application |
|---------|-------------|-------------|
| **Fixed Ratio Rule** | Cap interest at % of EBITDA | Default test: 30% of tax EBITDA |
| **Group Ratio Rule** | Allow higher deductions based on group position | Elective: Based on worldwide group's net interest/EBITDA |
| **De Minimis Threshold** | Small amounts exempt | Varies by jurisdiction (e.g., €3m in EU ATAD) |
| **Carry-Forward** | Unused capacity/disallowed interest | Typically 5-15 years |
| **Targeted Rules** | Address specific BEPS risks | Public benefit projects, third-party debt |

### Thin Capitalisation Rules: Three Main Approaches

| Approach | Mechanism | Typical Threshold | Countries Using |
|----------|-----------|-------------------|-----------------|
| **Debt-to-Equity Ratio** | Cap debt as multiple of equity | 1.5:1 to 4:1 | Australia (legacy), Japan, Korea |
| **EBITDA-Based** | Cap net interest as % of EBITDA | 20-30% | EU (ATAD), UK, Germany, Australia (2024+) |
| **Arm's Length** | Test if terms are commercial | Case-by-case | USA (§163(j)), OECD guidance |

### Debt Push-Down Structures

**The Concept**: Using debt at the target level (rather than parent level) to maximise interest deductions in high-tax jurisdictions.

```
BEFORE: Parent-Level Debt              AFTER: Debt Push-Down via SPV
┌─────────────────────┐                ┌─────────────────────┐
│   PARENT COMPANY    │                │   PARENT COMPANY    │
│   (Low-tax country) │                │   (Low-tax country) │
│                     │                │         │           │
│   Bank Loan ────────│                │    Equity only      │
│   Interest not      │                │         │           │
│   valuable here     │                │         ▼           │
│         │           │                │   ┌───────────┐     │
│    100% Equity      │                │   │    SPV    │     │
│         │           │                │   │(High-tax) │     │
│         ▼           │                │   │           │     │
│   ┌───────────┐     │                │   │ Bank Loan │     │
│   │  TARGET   │     │                │   │ Interest  │     │
│   │(High-tax) │     │                │   │ deductible│     │
│   │           │     │                │   │     │     │     │
│   │ No debt   │     │                │   │     ▼     │     │
│   │ No shield │     │                │   │  TARGET   │     │
│   └───────────┘     │                │   └───────────┘     │
└─────────────────────┘                └─────────────────────┘

Tax Shield: NONE                       Tax Shield: Interest × Tax Rate
```

### Key Requirements for Effective Debt Push-Down

| Requirement | Explanation | Exam Point |
|-------------|-------------|------------|
| **SPV in High-Tax Jurisdiction** | Locate borrowing entity where deductions are valuable | Choose jurisdiction with high corporate tax rate |
| **Tax Consolidation/Grouping** | Allow losses to offset target profits | Check if jurisdiction permits consolidation |
| **Sufficient Equity Base** | Meet thin cap thresholds | Ensure debt-to-equity complies with local rules |
| **Commercial Purpose** | Defend against GAAR challenges | Document business rationale beyond tax |
| **Arm's Length Terms** | Interest rate must be market rate | Benchmark against comparable third-party loans |

## 7.5 Worked Examples

### Example 7.1: EBITDA-Based Interest Limitation (Based on June 2025 Q7)

**Scenario**: OilCo Ltd, a subsidiary operating in Country X (30% EBITDA limit applies), has the following financial data:

| Item | Amount |
|------|--------|
| Operating profit before interest | $80 million |
| Depreciation | $20 million |
| Interest expense (intercompany loan from parent) | $40 million |
| Interest income | $5 million |

**Required**: Calculate the deductible interest under the EBITDA limitation rule.

**Model Answer**:

**Step 1: Calculate Tax EBITDA**
```
Operating profit before interest:     $80 million
Add: Depreciation:                    $20 million
Add: Net interest expense:            $35 million
                                     ─────────────
Tax EBITDA:                          $135 million
```

**Step 2: Calculate Net Interest Expense**
```
Gross interest expense:               $40 million
Less: Interest income:               ($5 million)
                                     ─────────────
Net interest expense:                 $35 million
```

**Step 3: Apply the 30% EBITDA Limit**
```
Maximum deductible interest = Tax EBITDA × 30%
                           = $135 million × 30%
                           = $40.5 million
```

**Step 4: Determine Result**
```
Net interest expense:                 $35 million
Maximum deductible:                   $40.5 million

Since $35m < $40.5m, ALL interest is deductible.
Disallowed interest: NIL
```

**Examiner's Note**: Always show the comparison clearly. If interest exceeds the cap, show:
- Deductible amount = $40.5 million
- Disallowed amount = Net interest - Cap
- Mention carry-forward availability if relevant

### Example 7.2: Thin Capitalisation Analysis (Based on December 2023 Q7)

**Scenario**: PetroCo has the following balance sheet position:

| Item | Amount |
|------|--------|
| Related-party debt (from parent) | $600 million |
| Third-party debt (bank loan) | $200 million |
| Share capital | $100 million |
| Retained earnings | $100 million |
| Interest rate on related-party debt | 8% |
| Arm's length rate | 6% |

Country Y applies:
- Debt-to-equity ratio limit: 3:1 (related-party debt only)
- Arm's length requirement for interest rates

**Required**: Calculate the deductible interest expense.

**Model Answer**:

**Step 1: Debt-to-Equity Ratio Test**
```
Related-party debt:                   $600 million
Total equity ($100m + $100m):         $200 million

Actual ratio: 600 ÷ 200 = 3:1
Permitted ratio: 3:1

RESULT: At the limit - no disallowance under D/E test
```

**Step 2: Arm's Length Interest Rate Test**
```
Actual intercompany interest rate:    8%
Arm's length benchmark rate:          6%
Rate differential:                    2%

Related-party debt principal:         $600 million
Excess interest = $600m × 2%:         $12 million

DISALLOWED (arm's length):            $12 million
```

**Step 3: Calculate Deductible Interest**
```
Gross related-party interest ($600m × 8%):    $48 million
Less: Arm's length disallowance:             ($12 million)
                                             ─────────────
Deductible related-party interest:            $36 million

Third-party interest ($200m × assumed rate):  As incurred (subject to EBITDA test)
```

**Key Exam Points**:
- Test debt-to-equity FIRST, then arm's length rate
- Only related-party debt typically subject to thin cap
- Third-party debt usually subject to EBITDA limits, not D/E ratios
- Disallowed interest may be recharacterised as dividend (WHT implications)

### Example 7.3: Debt Push-Down Structure (Based on June 2025 Q6 / December 2024 Q5)

**Scenario**: GlobalOil plc (UK parent, 25% tax) is acquiring TargetCo (Country Z, 35% tax) for $1 billion. GlobalOil is considering two structures:

**Option A**: Parent-level financing
- GlobalOil borrows $800 million from banks
- GlobalOil acquires TargetCo for $1 billion (equity)

**Option B**: Debt push-down via SPV
- GlobalOil establishes SPV in Country Z
- SPV borrows $800 million from banks
- SPV acquires TargetCo for $1 billion
- SPV and TargetCo form tax consolidated group

Interest rate: 5% per annum. TargetCo generates $150 million operating profit annually.

**Required**: Compare the tax efficiency of both options.

**Model Answer**:

**Option A: Parent-Level Financing**
```
Annual interest expense:              $800m × 5% = $40 million
Interest deduction in UK:             $40 million × 25% = $10 million tax saving

TargetCo taxable profit:              $150 million
Tax payable by TargetCo:              $150m × 35% = $52.5 million

TOTAL GROUP TAX: $52.5m - $10m =      $42.5 million
```

**Option B: Debt Push-Down via SPV**
```
Annual interest expense (in SPV):     $800m × 5% = $40 million

Consolidated taxable income:
  TargetCo operating profit:          $150 million
  Less: SPV interest expense:        ($40 million)
                                     ─────────────
  Consolidated taxable income:        $110 million

Tax payable (Country Z):              $110m × 35% = $38.5 million

TOTAL GROUP TAX:                      $38.5 million
```

**Comparison**:
| Structure | Annual Tax | Tax Saving vs Option A |
|-----------|-----------|------------------------|
| Option A (Parent debt) | $42.5 million | - |
| Option B (Debt push-down) | $38.5 million | $4 million p.a. |

**Tax saving from debt push-down**: $4 million per annum

**Why it works**:
- Interest deducted at 35% (Country Z) instead of 25% (UK)
- Additional tax shield: $40m × (35% - 25%) = $4 million

**Key Conditions** (always mention in exam):
1. Tax consolidation must be available in Country Z
2. SPV must satisfy thin capitalisation rules
3. Commercial substance required to withstand GAAR challenge
4. Consider CFC rules in UK - may tax SPV interest income

### Example 7.4: PSC vs Tax/Concession Interest Treatment (Based on June 2025 Q6)

**Scenario**: EnergyCo is evaluating two opportunities:
- **Country A**: Tax/concession regime (30% corporate tax)
- **Country B**: PSC regime (cost oil 60%, profit oil split 40/60 contractor/state)

Both projects require $500 million capital investment, funded 70% debt ($350 million) at 6% interest.

**Required**: Explain how interest deductions differ between the regimes.

**Model Answer**:

**Country A: Tax/Concession Regime**
```
Annual interest expense:              $350m × 6% = $21 million

Treatment:
• Interest is a deductible business expense
• Subject to thin capitalisation rules
• Reduces taxable income directly

Tax saving: $21m × 30% = $6.3 million per annum
```

**Country B: PSC Regime**
```
Annual interest expense:              $350m × 6% = $21 million

Treatment:
• Financing costs typically EXCLUDED from cost oil
• Interest does NOT reduce contractor's entitlement
• Contractor bears full cost of debt service from profit oil share

Tax saving: NIL (unless PSC specifically permits)
```

**Comparison Table**:

| Aspect | Tax/Concession (Country A) | PSC (Country B) |
|--------|---------------------------|-----------------|
| Interest deductible? | Yes (subject to thin cap) | Generally No |
| Reduces tax base? | Yes | No |
| Annual tax saving | $6.3 million | Nil |
| Optimal financing | Debt advantageous | Equity may be preferred |

**Key Exam Point**: Under PSC regimes, the inability to recover financing costs as cost oil is a significant disadvantage for debt-financed projects. This explains why many PSC projects are equity-financed or use ring-fenced project finance structures.

**Strategies under PSC**:
- Negotiate specific financing cost recovery in PSC terms
- Use operating leases instead of debt (lease payments may be cost-recoverable)
- Consider equity financing to avoid non-deductible interest burden

## 7.6 Error Prevention

### Common Mistakes to Avoid

| Mistake | Why It's Wrong | Correct Approach |
|---------|---------------|------------------|
| Applying D/E ratio to third-party debt | Most thin cap rules only apply to related-party debt | Check whether rule applies to all debt or only related-party |
| Forgetting to net interest income | EBITDA rules use NET interest expense | Always deduct interest income from interest expense |
| Ignoring carry-forward | Disallowed interest may be usable in future years | Mention carry-forward availability and period |
| Missing arm's length analysis | Related-party loans must be at market rates | Always test interest rate against benchmarks |
| Assuming PSC = tax/concession | Interest treatment differs fundamentally | Clearly distinguish fiscal regime type |
| Forgetting tax consolidation requirements | Debt push-down requires group relief | Check if consolidation is available and conditions |

### Pre-Submission Checklist

```
□ Identified ALL financing arrangements in the scenario
□ Distinguished related-party from third-party debt
□ Applied BOTH debt-to-equity AND EBITDA tests (if both exist)
□ Tested arm's length interest rate for intercompany loans
□ Calculated disallowed interest with clear workings
□ Considered carry-forward implications
□ For debt push-down: checked consolidation availability
□ For PSC questions: noted exclusion from cost oil
□ Mentioned relevant anti-avoidance rules (CFC, GAAR)
□ Considered withholding tax on cross-border interest
```

## 7.7 Quick Reference Card

### Key Formulae

```
EBITDA Limit Test:
─────────────────
Maximum Interest = Tax EBITDA × Fixed Ratio %
Disallowed Interest = Net Interest Expense - Maximum Interest (if positive)

Tax EBITDA = Taxable Income + Interest Expense + Depreciation + Amortisation

Debt-to-Equity Test:
───────────────────
Actual Ratio = Related-Party Debt ÷ Total Equity
Excess Debt = Related-Party Debt - (Equity × Permitted Ratio)
Disallowed Interest = Excess Debt × Interest Rate

Debt Push-Down Tax Saving:
─────────────────────────
Annual Saving = Interest Expense × (High Tax Rate - Low Tax Rate)
```

### BEPS Action 4: Key Numbers

| Element | OECD Recommendation | Common Implementation |
|---------|--------------------|-----------------------|
| Fixed ratio corridor | 10% - 30% of EBITDA | Most countries: 30% |
| De minimis threshold | Country discretion | €1-3 million typical |
| Carry-forward period | Country discretion | 5-15 years |
| Group ratio available? | Recommended | Many countries offer |

When you see these phrases, apply interest deduction analysis:

| Trigger | Action Required |
|---------|-----------------|
| "Financed by debt/loan" | Analyse deductibility under all applicable rules |
| "Parent company loan" | Apply thin cap AND arm's length tests |
| "Highly leveraged" | Expect thin cap disallowance |
| "Acquiring target company" | Consider debt push-down opportunity |
| "PSC/production sharing" | Note likely exclusion from cost oil |
| "EBITDA" mentioned | Calculate the limitation |

## Summary: Chapter 7 Key Takeaways

1. **Interest deductions are valuable** - they reduce taxable income, making debt an attractive financing choice

2. **Thin capitalisation rules limit abuse** - most countries restrict related-party debt via D/E ratios or EBITDA caps

3. **BEPS Action 4 is the global standard** - 30% of EBITDA is the benchmark fixed ratio

4. **PSC regimes differ fundamentally** - financing costs typically excluded from cost oil

5. **Debt push-down creates value** - locating debt in high-tax jurisdictions maximises the tax shield

6. **Always check multiple limitations** - a single loan may face D/E, EBITDA, AND arm's length restrictions

7. **Anti-avoidance rules apply** - CFC rules, GAAR, and hybrid mismatch rules may override tax benefits
